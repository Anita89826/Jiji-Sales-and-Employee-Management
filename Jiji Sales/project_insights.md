## Overview
This SQL project demonstrates core database querying and analysis using a fictional company dataset, **Jiji**.  
It includes employees, customers, positions, and sales order data.

## Dataset Description
- **Employees:** Details of staff and their salaries  
- **Positions:** Job roles and IDs  
- **Customers:** Customer names, cities, and emails  
- **Orders:** Purchases and total amounts  

## Key Business Questions
- Retrieve customer names and cities  
- Combine order IDs and total amounts  
- Find top customers and highest-paid employees  

## SQL Techniques Used
`SELECT`, `JOIN`, `UNION`, `GROUP BY`, `ORDER BY`, and Aggregation Functions

## Key Insights
- Lagos customers contribute the highest number of orders.  
- Managers have the highest salary averages.  
- Multiple repeat customers indicate loyalty and sales growth.  

## Skills Demonstrated
SQL Querying | Data Analysis | Database Design | Business Insight Reporting  

## Recommendation
Focus marketing efforts in **Lagos** and reward loyal customers to increase revenue growth.
